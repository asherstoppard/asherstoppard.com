## Prerequisites

- Node/NPM
- Yarn

## Getting Started

Clone the repository

```sh
$ git clone git@github.com:asherstoppard/asherstoppard.dev.git
$ cd asherstoppard.dev
```

Install the dependencies

```sh
$ yarn
```

Run the application locally

```sh
$ yarn dev
```
