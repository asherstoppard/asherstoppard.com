## Prerequisites

- Node/NPM
- Yarn

## Getting Started

Clone the repository

```sh
$ git clone git@github.com:asherstoppard/asherstoppard.com.git
$ cd asherstoppard.com
```

Install the dependencies

```sh
$ yarn
```

Run the application locally

```sh
$ yarn dev
```
